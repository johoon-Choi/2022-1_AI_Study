## Data type
```python
a = 10
b = 5.6

print(" a is", a, type(a)) # a is 10 <class 'int'>
print(" b is", b, type(b)) # b is 5.6 <class 'float'>
```

## for loop
```python
for i in range(n):
   print(i)
# 0 ~ n-1 까지 출력
```

## while loop
```python
while True:
   print(num)
   num -= 1
   if num == 0:
      break
```
